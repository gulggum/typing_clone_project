# 🧠 타이핑 클론 앱

타자 연습을 위한 간단한 웹 앱
주어진 문장 3개를 타이핑하며 정확도와 분당 타자수(cpm)을 실시간으로 확인 할 수 있습니다.

---

## 주요 기능

- 실시간 입력 비교(올바른 글자/오타 색상 표시)
- 제한 시간 설정 및 타이머 기능
- 정확도(%) 및 cpm(분당 타수) 계산
- 문장 자동 전환 및 input입력 결과 출력
- 다시 시작 기능

---

## 사용 기술

- HTML
- CSS
- JavaScript (Vanilla JS)
